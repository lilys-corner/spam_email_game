## Game Description
This application, in the form of an educational game, is to raise awareness and educate players on risks of spam/phishing emails and how to identify spam email. 
There are two modes of game play:
- Game
  - Player identifies if the mail is spam or not, results presented at the end with correct answers if score less than 100
- Quiz
  - Player takes a multiple choice quiz testing their knowledge on spam/phishing emails, results presented at the end with correct answers if score less than 100
### Other Game Features
- Log-In
  - At every start up, users are prompted to log-in or create an account.
  - Users also have the option to use guest mode. **(Warning guest mode does not save data)**
- Leaderboard
  - Displays the top 10 registered users scores for both Game and Quiz
- Settings
  - Users have the ability to customize these settings:
    - Windowed/Full Screen
    - Volume
    - Font Size
    - Dyslexia Font
